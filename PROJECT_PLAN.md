# Живой Лум

## О проекте

Живой Лум — современная социальная сеть для жителей села Лум.

- 📰 Новости
- 📷 Фотоальбомы
- 🎥 Видео
- 💬 Общение жителей
- 📢 Объявления
- 📅 Мероприятия
- 🗳️ Голосования
- 🏛️ История села
- 📍 Карта села
- 👥 Профили жителей
- 🔔 Уведомления
- 👨‍💼 Администрирование
